//// Travel Expense Tracker

A mobile application to track expenses during trips.

This project was created from a personal need to easily record travel expenses, even when there is no internet connection. The main goal is simplicity, organization, and continuous learning in mobile development.

_________


//// About the project

**Travel Expense Tracker** is a mobile app built with React Native and Expo, using SQLite for local data storage.

This is a personal project and part of my learning journey in mobile development, component architecture, and local persistence. The app is constantly evolving as new ideas and improvements come up.

__________

## Technologies used

- React Native  
- Expo  
- JavaScript  
- SQLite  
- Node.js  

__________

## Features

- Create and manage expenses
- Edit and delete records
- Expense categories
- Local storage (works offline)
- Clean and simple interface
- Android and iOS support

---

## Running the project

git clone https://github.com/DiegoGamaDev/travel-expense-tracker.git
cd travel-expense-tracker
npm install
npx expo start
